Presentation of SolarPanelDashboard Project

Project Overview

This project was a three-part assignment to complete the final project of a Level 6 Diploma in Software Development. The assignment consisted of:

  1. The project proposal, which included team organization, requirements analysis, design documentation, planning process, and client/lecturer meetings.
  2. The project execution, involving project management, software development, application security, testing, and client/lecturer feedback.
  3. Project presentation, which is the purpose of this website.

The Solution

Our solution is a comprehensive dashboard designed specifically for a Solar Panel Installation company. It provides a centralized platform to manage various aspects of the business.

Key Features

  • Secure login system for authorized access to the dashboard.
  • Real-time weather information for accurate decision-making.
  • Inventory management with functionalities for adding, deleting, and editing stock items.
  • User management for administrators to manage user profiles and details.
  • Future enhancements planned based on a roadmap.
