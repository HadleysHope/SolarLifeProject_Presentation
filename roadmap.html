Roadmap

Here is the roadmap for future enhancements and features planned for the Solar Panel Dashboard:

Feature 1: Home Dashboard Company Notice Board

Turn the Home Section into a company notice board with Edit function for approved Users to be able to update the possible following:

  • Company Announcements
  • Ongoing Tasks: Project News
  • Performance Metrics: Sales Figures/Customer Satisfaction Rating
  • Employee Recognition: Employee of the Month
  • Upcoming Events
  • Internal Resources: Links to Employee Handbook, Code of Conduct, Safety Guidelines
  • Industry Updates
  • Customer Reviews

Feature 2: SMTP Email

Sign up to SMTP Provider to have real Email Messaging services to users

Feature 3: 5 Day Weather Forecast

Upgrade Weather API from current Weather to 5 Day forecast

Feature 4: Authentication

Implement WebToken Authentication callback from front-backend instead of just backend to database

Feature 5: User Profile Settings

Create Profile Features for Users and allow changing password

Feature 6: CRM (Customer Relationship Manager)

Allow company to manage customer profiles from the dashboard

Feature 7: Scheduling Console + Mobile App

Manage Installer schedules and create Mobile App for rosters and jobs integrated to app

Feature 8: Reports

Generate reports based on dashboard information

Feature 9: Upgraded Stock Inventory

More Features for Stock Management such as:

  • Ordering Emails Generated on reaching Minimum Stock Level
  • Integration with Suppliers
